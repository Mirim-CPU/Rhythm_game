- 떨어지는 노트를 정해진 박자에 누르는 **리듬 게임**입니다.
- **C언어**를 이용한 **콘솔 게임**입니다.

## 플레이 방법
처음 게임을 실행시키면, 게임에 사용할 **키 설정**을 할 수 있습니다.

기본적으로 **4키** 기반입니다. 쓰고자 하는 키를 순서대로 한 번씩 눌러 주세요.

맵은 **프로그램 경로/maps** 폴더에 저장합니다.

원하는 대로 맵 폴더와 맵 데이터를 넣어서 플레이가 가능합니다.
